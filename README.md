### 使用PHP框架 YII 1.1.15 版本
    YII框架由2008年发展至今已经相当成熟，2014-10-12刚发布了2.0的正式版，1.1版本官方会继续维护。
    YII框架在百度和腾讯大量的使用，无论从安全性，还是开发效率上都值得推荐，所以该项目选择使用YII框架。
